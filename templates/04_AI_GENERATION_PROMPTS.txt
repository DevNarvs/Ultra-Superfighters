================================================================================
  TEMPLATE 04: AI GENERATION PROMPTS (Frame-by-Frame)
  Super Battlemancers
================================================================================
  PURPOSE:  Ready-to-copy-paste AI image generator prompts for EACH animation
            and skill effect, individually. Generate one animation at a time.
  INPUT:    <- 01_CHARACTER_DESCRIPTION.txt (Visual Identity, Palette)
            <- 02_CHARACTER_DESIGN.txt (Pose Guide, Tech Specs)
            <- 03_CHARACTER_SKILLS_EFFECTS.txt (Skill Details, Effect Specs)
  OUTPUT:   Individual sprite strips per animation row & per skill effect
================================================================================


HOW TO USE THIS TEMPLATE
================================================================================

1. Fill in ALL [bracket] placeholders using data from templates 01, 02, and 03.
2. Each PROMPT section below is a complete, self-contained copy-paste block.
3. Copy ONE prompt at a time into your AI image generator.
4. After generating all strips, STITCH them together:
   - Animation rows 0-10 → stack vertically → full character sprite sheet
   - Skill effects → keep as individual horizontal strip files

STITCHING GUIDE (for character sprite sheet):
   Use Photopea.com (free), Photoshop, or GIMP:
   - Create new canvas: 384 x 704 pixels, transparent background.
   - Paste Row 0 at position (0, 0).
   - Paste Row 1 at position (0, 64).
   - Paste Row 2 at position (0, 128).
   - ... Row N at position (0, N * 64).
   - Row 10 at position (0, 640).
   - Save as PNG with transparency.
   - Use "Nearest Neighbor" resampling when resizing to preserve pixel art.

IMPORTANT: If generating at 2x scale (128x128 per frame), adjust all pixel
values accordingly (multiply by 2). Final sheet at 2x = 768 x 1408 pixels.

================================================================================
TIPS FOR AI IMAGE GENERATION
================================================================================

1. STYLE CONSISTENCY:
   If your AI tool supports image references, upload your character's idle
   pose (or the Fire Battlemancer sprite) as a style reference when generating
   each new strip. This keeps the art style consistent across all animations.

2. IF AI GIVES INDIVIDUAL IMAGES (not a strip):
   Generate each frame as a separate image, then stitch them side-by-side:
   - Open Photopea.com (free, browser-based Photoshop)
   - Create new canvas at the exact total strip size
   - Paste each frame at the correct X offset:
     Frame 0 at x=0, Frame 1 at x=[frame_width], Frame 2 at x=[frame_width*2]...
   - Use "Nearest Neighbor" resampling to preserve pixel art

3. SCALE NOTES:
   - If your AI generates at higher resolution, resize DOWN using
     "Nearest Neighbor" interpolation (NOT bilinear/bicubic).
   - If generating at 2x scale, all pixel values in the prompts should be
     doubled (e.g., 64x64 becomes 128x128, y=58 becomes y=116).

4. BACKGROUND REMOVAL:
   - Use remove.bg, Photopea "Magic Wand + Delete", or GIMP "Color to Alpha"
   - Make sure the background is truly transparent (not white)

5. COMMON AI MISTAKES TO WATCH FOR:
   - 3/4 view instead of strict side view → re-prompt with emphasis on "SIDE"
   - Front-facing character → re-prompt: "NOT front-facing, side profile only"
   - Added effects/weapons → re-prompt: "body and clothing ONLY, no effects"
   - Inconsistent proportions between frames → use same prompt structure
   - Frames not aligned → manually align in Photopea before stitching

6. PROMPT STRENGTHENING:
   If the AI ignores certain rules, try moving those rules to the BEGINNING
   of the prompt and repeating them. Example: Start with "CRITICAL: This must
   be a strict SIDE VIEW profile..." and repeat at the end.
