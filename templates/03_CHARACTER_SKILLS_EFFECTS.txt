HOW HYBRID EFFECTS WORK
================================================================================

  Your Sprite Art    +    Engine Particles    =    Final In-Game Look
  (shape & style)         (glow, trail, sparks)    (polished & dynamic)

You draw the CORE shape. The engine adds glow, trails, sparks at runtime.
Your sprites do NOT need trailing particles or ambient glow — just the main shape.

GENERAL RULES (ALL EFFECTS)
================================================================================
Layout            : Horizontal strip — all frames in a single row, left to right
Background        : Transparent (PNG-24 with alpha)
Art style         : Must match the character sprite style (anime pixel art)
Outline           : 1px dark outline
Shading           : 2-3 levels per color
Direction         : Projectiles face RIGHT (engine flips for left)
Centering         : Effect centered within each frame
No extras         : No trailing particles or glow — engine adds those

TIPS FOR AI IMAGE GENERATION
================================================================================

1. If AI gives individual images instead of a sprite sheet:
   Generate each frame separately, then stitch side-by-side in
   Photopea.com (free browser Photoshop) or even MS Paint.

2. To remove backgrounds:
   Use remove.bg, or Photopea "Magic Wand + Delete",
   or GIMP "Color to Alpha".

3. To resize to exact dimensions:
   Photopea → Image → Canvas Size → set exact pixels.
   Use "Nearest Neighbor" resampling to preserve pixel art.

4. To keep style consistent:
   Upload your character sprite as a style reference
   (if your AI tool supports image references).

5. Frame alignment:
   If frames are different sizes, create new canvas at total size,
   paste each frame at correct X position:
   Frame 0 at x=0, Frame 1 at x=frameWidth, Frame 2 at x=frameWidth*2...
